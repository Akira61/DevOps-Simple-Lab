# Python Calculator CI Pipeline

A simple Python calculator project demonstrating **Unit Testing** and **CI/CD automation** using GitHub Actions.

---

```txt
calculator-project/
│
├── calculator.py

    └── workflows/
        └── python-tests.yml
```

---

# Features

- Basic calculator functions
- Automated unit testing
- GitHub Actions CI pipeline
- Automatic test execution on every push

---

# Technologies Used

- Python 3.9
- Git & GitHub
- GitHub Actions
- Unit Testing

---

# How It Works

The project uses GitHub Actions to automatically run tests whenever code is pushed to the repository.

Workflow process:

1. Push code to GitHub
2. GitHub Actions starts automatically
3. Python environment is created
4. Unit tests are executed
5. Pipeline passes or fails based on test results
